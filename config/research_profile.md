# 研究プロファイル

このファイルは daily_digest.py が新着論文の関連度を判定する際の「ものさし」として使われます。
修士論文(浦部蒼太, 2023年度「Hαフレア: Flares on close binary system probed by Hα line profiles」)
および既発表論文(Urabe et al. 2026, PASJ 78, 310 / arXiv:2603.01095)の内容を元に作成しています。
論文が増えた場合は scripts/build_profile.py を再実行して更新してください。

## 主要な研究テーマ
- RS CVn型連星(HR 1099, UX Arietisなど)における恒星フレア・スーパーフレアの観測的研究
- フレアに伴うHα輝線プロファイルの時間変化(非対称性・視線速度シフト)からの発生領域推定
- X線(MAXI等)と可視光分光(Seimei望遠鏡/SCAT)を組み合わせた多波長・追観測(フォローアップ)研究
- 連星の軌道位相(自転・食)を利用したフレア領域のジオメトリ(位置・スケール)の制約
- フレアのエネルギー収支(光学: 〜10^32 erg、X線: 10^36〜10^39 erg規模)の議論

## よく使う観測手法・データソース
- MAXI(全天X線監視装置)による突発天体現象(フレア)のトリガー検出
- Seimei望遠鏡(3.8m、京都大学)+ SCAT分光器による多夜・高頻度の分光モニタリング
- Hα輝線プロファイルの抽出・フィッティングによる視線速度変調の解析
- 連星の軌道位相(食の有無)を利用したフレア領域の可視・不可視タイミングからの幾何学的制約
- (将来的な拡張の可能性)NICER, XMM-Newton等、他のX線衛星データとの組み合わせ

## 関連度が高い/低いの目安
- **高い**: RS CVn型連星や活動的な晩期型星のフレア・スーパーフレアに関する観測研究、
  Hα等の輝線プロファイルを用いたフレア領域推定、MAXIやSeimei望遠鏡(あるいは類似の
  X線モニター+地上分光フォローアップ)を用いた観測キャンペーン、フレアに伴う
  質量放出(CME的振る舞い)の検出、連星系における恒星活動の研究
- **中間程度**: 太陽フレア観測研究で恒星への応用が議論されているもの、系外惑星への
  宇宙天気・フレアの影響、単独星(連星でない)のスーパーフレア統計研究
- **低い**: 銀河・宇宙論・重力波・系外惑星の存在検出など、恒星の磁気活動やフレア現象と
  直接関係しない分野
